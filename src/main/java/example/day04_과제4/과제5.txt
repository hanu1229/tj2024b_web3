JPA 과제5 : 비품관리 CRUD
[기능구현]
    과제4 서비스에서 사용한 save(), findById(), findAll(), deleteById(), existById 등등
    미리 만들어진 함수를 사용하지 않고 네이티브쿼리 추상메소드를 생성하시오

    01. 비품 등록 기능
            - 사용자는 새로운 비품 정보를 등록 할 수 있어야 한다.
            - 등록 시 필요한 정보 : 비품명(name), 비품 설명(description), 수량(quantity)
            - 등록 시간(createdDate)과 수정 시간(modifiedDate)은 자동으로 기록되어야 한다.
    02. 비품 전체 조회 기능
        - 저장된 모든 비품 목록을 조회할 수 있어야 한다.
    03. 비품 개별 조회 기능
        - 특정 비품의 상세 정보를 id를 통해 조회 할 수 있어야 한다.
    04. 비품 수정 기능
        - 수정 시 특정 비품의 id를 기준으로 수정하며, 수정 시간은 자동으로 갱신되어야 한다.
    05. 비품 삭제 기능
        - 특정 비품의 id를 기준으로 해당 비품 정보를 삭제할 수 있어야 한다.
